# 📚 아두이노 기본 라이브러리 버전 관리 시스템

> 효율적인 라이브러리 버전 관리

## 🗂 파일 구조

### `_file_meta.json`
   📄 라이브러리 버전 정보를 저장하는 JSON 파일   
  
  #### 수작업으로 변경해야 함.

  #### 중요 항목
   -. header -> updated_at : 메타파일 수정일자  
   -. body -> created_at   : 해당 라이브러 수정일자  
   -. body -> name, version, url : 아두이노 라이브러리.json 파일에서 가져옮
